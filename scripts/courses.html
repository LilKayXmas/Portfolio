Human Computer Interactions

This course involves the examination of factors influencing the usability of a computer system. This course discusses best practices and procedures for designing interfaces. It also focuses on the underlying principles of interaction process with computers.

Human Computer Interactions
A discipline concerned with the design, evaluation and implementation of interactive computing systems for human use and with the study of the major phenomena surrounding them
Safety
User interfaces that affect life-or-death decisions
Utility
What services a system provides
Effectiveness
User's ability to achieve goals

Needs Assess & Tech Transfer

This course builds student skills in two different yet interacting areas: needs assessment (requirements analysis) and group problem solving.

Network-Based Multimedia

This course presents fundamental topics of designing and implementing multimedia on the internet. Each topic is presented along with the underlying computer technology that supports it and hands-on projects incorporating the concepts.

Protocols
Set of rules that define how a client workstation can communicate with a server
Uniform Resiurce Locator
The address of a document or other file accessible on the internet
Domain
An area of the Internet of a particular organization or person manages
